### 使用 datetime 模块
```python
from datetime import datetime

# 获取当前日期
date = datetime.now()
# 当前是周几
day = date.isoweekday()
# 当前是几点(24小时)
hour = date.hour
# 将日期格式化：方式1
date.strftime('%Y-%m-%d %H:%M:%S')
# 将日期格式化：方式2
date_str = '%s-%s-%s' % (date.year, date.month, date.day)
```
输出结果：
```python
# 当前日期'.'后面是微秒，即date.microsecond
2017-09-12 09:43:00.400000
# 周四
4
# 上午9点
9
# 格式化后的日期：方式1
2017-09-12 09:43:00
# 格式化后的日期：方式2。区别就是方式1格式化后的月份小于10的时候自己默认补0
2017-9-12
```
