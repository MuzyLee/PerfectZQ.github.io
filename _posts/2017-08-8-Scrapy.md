### 创建scrapy项目
```
# 安装scrapy
pip install scrapy

# 进入要创建项目的文件夹
# shift + 右键 在当前文件夹打开命令窗口

# 需要提前配置好python的环境变量
scrapy startproject mySpider
```
### scrapy的项目结构
```
mySpider/
   scrapy.cfg            # 项目的配置文件
   mySpider/
       __init__.py
       items.py
       pipelines.py
       settings.py
       spiders/          # 用户自己写的spider要放在spiders目录下面
           __init__.py
           ...
```
